写了一个简易的抢课程序，欢迎大家测试交流。

访问：GitHub copycat 获取源码

本程序仅供学习研究，如用于其他用途概不负责。

总体思路采用 selenium 模拟浏览器（Firefox）操作，进行自动化抢课，re 判断抢课成功，跳出循环

存在问题：网络加载缓慢，会有一定几率报错。

运行环境：

Python 3、Firefox浏览器（需配置 geckodriver，自行百度）

Python 3 安装 selenium 包

执行方法：

打开 copycat.py F5 运行即可
